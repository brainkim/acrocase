# acrocase

A minimal Shovel application, built with [Shovel](https://github.com/bikeshaving/shovel).

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm start` - Run production build
